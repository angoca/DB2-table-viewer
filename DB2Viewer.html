DB2 Viewer Applet

Your Internet browser does not support Java applets.

If you have problems, execute this from the command line:

appletviewer -J-Djava.security.policy=http://angoca.github.com/DB2-table-viewer/java.policy.applet http://angoca.github.com/DB2-table-viewer/DB2Viewer.html

In Windows, if you do not have Java installed, you can use the Java included in DB2.

cd C:\Program Files\IBM\SQLLIB\java\jdk\bin
